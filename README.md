## Authors

- [@ferryu46](https://github.com/ferryu46/)


## Installation

Clone this repository

```bash
  git clone https://github.com/ferryu46/ForumAPIv2.git .
  cd ForumAPIv2
```

Install project dependencies

```bash
  npm install
```

Run locally

```bash
  npm run start
```
